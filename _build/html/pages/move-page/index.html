Move a page¶

You can always move a page to another position on the same level (within the parent node). It’s also possible to move a node (one page or a page with all sub pages) to any position in the navigation tree. See this page for more information: Edit Page Navigation

Do the following to move a page within the same level:

  1. Open the page in Design mode.

The navigation structure with the existing pages are now opened to the left:

If you’re already editing the page or the navigation won’t open automatically, you can click here to open the navigation:

  1. Select the page you want to move.
  1. Use the triangles to move the page up or down.
  1. Save when you’re done.
