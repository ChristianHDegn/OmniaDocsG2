Dashboard¶

Here you can build the Dashboard to your needs. You set up the Dashboard much like you would do a page type. Most sections and blocks can be used.

As you would a page type, the first step is to add the layout, using sections. See this page for more information: Working with section and blocks

You then add the blocks you need. All available blocks are listed on this page: Blocks

The Metrics block can be especially useful: The Metrics block

You can also edit some of the metrics settings, in Omnia Admin, see: Metric settings
