Using the Omnia Block web partÂ¶

Some of the blocks in Omnia can be used in any Team site, including Community sites. To add a block there, you use the Omnia Block web part.

This is how:

  1. Go to the Team site.
  2. Go to the page in the Team site.
  3. Edit the page.
  1. Click on the web part that should be above the block.
  2. Click the plus.
  1. Select the Omnia Block web part.
  1. Select block.
  1. Click the pen to edit settings for the block.

The settings are shown, for example:

You use the settings for the block exactly the same way as when the block is placed on an Omnia page. See the description for the block here: Blocks

Note! In an environment that has been upgraded from an earlier version of Omnia, an older web part can still be available, just called Omnia. You should not use that web part any more.
